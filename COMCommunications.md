Standard of COM Communications
======
Standards
------
1. First two Characters are command
2. Commands terminated by carriage return

To Arduino
======

Set Frequency
------
`F <frequency in Hz>`

Oscillator ON
------
`ON`

Oscillator OFF
------
`OF`

Set Start Frequency
------
`L <frequency in Hz>`

Set Stop Frequency
------
`H <frequency in Hz>`

Set Interval
------
`I <frequency in Hz>`

Start Scan
------
`S`

To PC
======

Send Data for Freq
------
`D <frequency in Hz> <SWR> <Voltage> <Resistance>`
